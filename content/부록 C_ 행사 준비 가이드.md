## 👥 조직 구성

### 리더십 구조

```
나팔단장 (1명)
├── 부단장 A (1명) - 교육/리허설 담당
├── 부단장 B (1명) - 행정/물류 담당
│
├── 섹션 A 리더 (1명) → 단원 30명
├── 섹션 B 리더 (1명) → 단원 30명
├── 섹션 C 리더 (1명) → 단원 30명
├── 섹션 D 리더 (1명) → 단원 30명
│
└── 지원팀
    ├── 악기 관리 (2명)
    ├── 음향 담당 (1명)
    ├── 영상 기록 (1명)
    └── 의료 지원 (1명)
```

### 역할별 책임

| 역할 | 주요 책임 |
|------|----------|
| **단장** | 전체 총괄, 지휘, 최종 결정 |
| **부단장 A** | 교육 프로그램, 리허설 진행 |
| **부단장 B** | 일정 관리, 물류, 행정 |
| **섹션 리더** | 30명 관리, 출석, 진도 체크 |
| **악기 관리** | 악기 점검, 수리, 배분 |
| **음향 담당** | 사운드 체크, 음향 장비 |
| **영상 기록** | 사진/영상 촬영, 아카이브 |
| **의료 지원** | 응급 상황 대응, 건강 관리 |

### 비상 연락망

|------|------|--------|------|

| 부단장 A | ________ | ________ | 현장 |
| 부단장 B | ________ | ________ | 행정 |
| 의료 담당 | ________ | ________ | 응급 |
| 현지 가이드 | ________ | ________ | 통역 |
| 대사관 | ________ | ________ | 긴급 |

### 비상 상황별 대응

#### 의료 응급

| 상황 | 대응 |
|------|------|
| 경미한 부상 | 의료 담당 처치, 계속 참여 여부 판단 |
| 심각한 부상 | 즉시 119, 병원 이송, 대사관 연락 |
| 탈진/실신 | 안전한 곳 이동, 수분 공급, 휴식 |
| 알레르기 반응 | 에피펜 사용, 병원 이송 |

#### 악기 문제

| 상황 | 대응 |
|------|------|
| 악기 파손 | 예비 악기 즉시 교체 |
| 악기 분실 | 예비 악기 사용, 분실 신고 |
| 다수 악기 문제 | 섹션 재편성, 축소 연주 |

#### 인원 문제

| 상황 | 대응 |
|------|------|
| 단원 불참 | 섹션 내 재배치 |
| 다수 불참 | 섹션 통합, 프로그램 조정 |
| 단장 유고 | 부단장 A가 지휘 대행 |

#### 환경 문제

| 상황 | 대응 |
|------|------|
| 악천후 | 실내 대체 장소 이동 |
| 음향 문제 | 어쿠스틱 연주 전환 |
| 일정 변경 | 즉시 전체 공지, 유연 대응 |

## 📊 평가 및 피드백

### 행사 후 평가 항목

| 영역 | 평가 항목 | 점수 (1-5) |
|------|----------|:----------:|
| **연주 품질** | 소리의 정확성 | ☐ |
| | 앙상블 조화 | ☐ |
| | 다이내믹 표현 | ☐ |

| | 소통 원활성 | ☐ |
| | 문제 대응 | ☐ |
| | 안전 관리 | ☐ |
| **단원 만족** | 교육 프로그램 | ☐ |
| | 리더십 | ☐ |
| | 전체 경험 | ☐ |

### 단원 피드백 설문 (예시)

1. 전체적인 만족도는? (1-5)
2. 가장 좋았던 점은?
3. 개선이 필요한 점은?
4. 교육 프로그램에 대한 평가는?
5. 리더십에 대한 평가는?
6. 다음 행사 참여 의향은?
7. 기타 의견

### 결과 보고서 구성

1. **개요**: 행사 기본 정보
2. **참가자**: 인원, 구성
3. **프로그램**: 진행 내용
4. **성과**: 목표 달성도
5. **문제점**: 발생 이슈
6. **교훈**: 배운 점
7. **제언**: 향후 개선 사항
8. **부록**: 사진, 영상, 자료
