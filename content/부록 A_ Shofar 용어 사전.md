# 부록 A: Shofar 용어 사전

> 쇼파르 교육에서 사용되는 주요 용어들의 정의와 설명

---

## 히브리어 용어

### 악기 관련

| 용어 | 히브리어 | 발음 | 정의 |
|------|---------|------|------|
| **Shofar** | שׁוֹפָר | 쇼파르 | 동물 뿔로 만든 고대 관악기 |
| **Keren** | קֶרֶן | 케렌 | 뿔 (일반적 의미) |
| **Chatzotzrah** | חֲצוֹצְרָה | 하초츠라 | 금속 트럼펫 (쇼파르와 구분) |

### 소리 관련

| 용어 | 히브리어 | 발음 | 정의 |
|------|---------|------|------|
| **Tekiah** | תְּקִיעָה | 테키아 | 단일 긴 소리 |
| **Shevarim** | שְׁבָרִים | 쉐바림 | 3개의 짧은 소리 ("부서진 것들") |
| **Teruah** | תְּרוּעָה | 테루아 | 9개 이상의 빠른 소리 ("경보") |
| **Tekiah Gedolah** | תְּקִיעָה גְּדוֹלָה | 테키아 게돌라 | 매우 긴 소리 ("큰 테키아") |
| **Kolot** | קוֹלוֹת | 콜롯 | 소리들 (복수형) |

### 역할 관련

| 용어 | 히브리어 | 발음 | 정의 |
|------|---------|------|------|
| **Ba'al Tekiah** | בַּעַל תְּקִיעָה | 바알 테키아 | 쇼파르 부는 자 |
| **Makri** | מַקְרִיא | 마크리 | 소리 이름을 부르는 자 |
| **Tokea** | תּוֹקֵעַ | 토케아 | 나팔 부는 자 (Ba'al Tekiah와 동의어) |

### 절기 관련

| 용어 | 히브리어 | 발음 | 정의 |
|------|---------|------|------|
| **Rosh Hashanah** | רֹאשׁ הַשָּׁנָה | 로쉬 하샤나 | 유대 새해 |
| **Yom Kippur** | יוֹם כִּפּוּר | 욤 키푸르 | 속죄일 |
| **Elul** | אֱלוּל | 엘룰 | 유대력 6번째 달 (회개의 달) |
| **Tishrei** | תִּשְׁרֵי | 티쉬레이 | 유대력 7번째 달 |
| **Yovel** | יוֹבֵל | 요벨 | 희년 (50년째 해) |

### 종교 개념

| 용어 | 히브리어 | 발음 | 정의 |
|------|---------|------|------|
| **Mitzvah** | מִצְוָה | 미츠바 | 계명, 선행 |
| **Teshuvah** | תְּשׁוּבָה | 테슈바 | 회개, 돌아옴 |
| **Halakha** | הֲלָכָה | 할라카 | 유대 율법 |
| **Akedah** | עֲקֵדָה | 아케다 | 이삭 번제 사건 ("결박") |
| **Bracha** | בְּרָכָה | 브라카 | 축복 |

## 음악/연주 용어

### 기본 용어

| 용어 | 정의 |
|------|------|
| **Embouchure** | 마우스피스에 입술을 위치시키는 방법 |
| **Aperture** | 입술 사이의 공기가 통과하는 구멍 |
| **Lip Buzzing** | 입술만으로 윙윙거리는 소리 내기 |
| **Mouthpiece** | 입술이 닿는 악기의 부분 |

### 호흡 관련

| 용어 | 정의 |
|------|------|
| **Diaphragmatic Breathing** | 횡격막 호흡 |
| **Breath Support** | 호흡 지지 |
| **Airflow** | 공기 흐름 |
| **Breath Control** | 호흡 조절 |

### 음색/다이내믹

| 용어 | 정의 |
|------|------|
| **Tone** | 음색 |
| **Resonance** | 공명 |
| **Projection** | 소리의 투사 |
| **Dynamics** | 음량의 변화 |
| **Crescendo** | 점점 크게 |
| **Decrescendo** | 점점 작게 |
| **Fortissimo (ff)** | 매우 세게 |
| **Pianissimo (pp)** | 매우 여리게 |

### 앙상블 용어

| 용어 | 정의 |
|------|------|
| **Ensemble** | 앙상블, 합주 |
| **Synchronization** | 동기화 |
| **Unison** | 유니즌, 동음 연주 |
| **Conductor** | 지휘자 |
| **Cue** | 신호 |
| **Cutoff** | 종료 신호 |
