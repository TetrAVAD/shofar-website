# 🎺 Shofar 전문가 교육 커리큘럼

> **프로그램명**: Shofar Master Program - 나팔단장 양성 과정
> **대상**: 이스라엘 행사 참여 나팔수 120명
> **목표**: 전문가(Ba'al Tekiah) 수준의 쇼파르 연주자 양성

🌐 **Language**: [[English_Curriculum/00_Curriculum_Overview|🇺🇸 English]] | [[Taiwanese_Curriculum/00_Curriculum_Overview|🇹🇼 繁體中文]]

## 📋 커리큘럼 구조 개요

본 커리큘럼은 **6개 모듈**로 구성되며, 각 모듈은 이론과 실기를 균형 있게 포함합니다. 전체 과정을 이수하면 쇼파르의 역사적, 신학적, 기술적 측면을 모두 아우르는 전문가로 성장할 수 있습니다.

### 모듈 구성

| 모듈 | 제목 | 핵심 내용 | 예상 시간 |
|------|------|----------|----------|
| **Module 1** | 쇼파르의 기원과 역사 | 성경적 배경, 유대 전통, 현대적 의미 | 4시간 |
| **Module 2** | 쇼파르의 종류와 구조 | 재료, 제작, 선택 기준, 관리법 | 3시간 |
| **Module 3** | 기초 연주 기법 | 호흡법, 앙부슈르, 기본 소리 생성 | 6시간 |
| **Module 4** | 전통 소리와 의식 | Tekiah, Shevarim, Teruah, 의식 순서 | 6시간 |
| **Module 5** | 고급 기법과 앙상블 | 음색 조절, 다이내믹, 단체 연주 | 8시간 |
| **Module 6** | 영적 준비와 리더십 | 연주자의 마음, 단장의 역할, 행사 준비 | 5시간 |

**총 교육 시간**: 약 32시간

---

## 🎯 학습 목표

### 지식 (Knowledge)
- 쇼파르의 3,000년 역사와 성경적 맥락 이해
- 다양한 쇼파르 종류와 각각의 특성 파악
- 유대 전통에서의 쇼파르 의식 절차 숙지

### 기술 (Skills)
- 4가지 전통 소리(Tekiah, Shevarim, Teruah, Tekiah Gedolah) 완벽 구사
- 안정적인 호흡과 앙부슈르 유지
- 120명 단체 연주 시 조화로운 합주 능력

### 태도 (Attitude)
- 쇼파르 연주의 영적 의미와 책임감 인식
- 겸손과 경건함으로 임하는 자세
- 단원들과의 협력과 조화

## 📅 추천 학습 일정

### 집중 과정 (2주)
- 1주차: Module 1-3 (이론 + 기초 실기)
- 2주차: Module 4-6 (심화 실기 + 리더십)

### 정규 과정 (8주)
- 1-2주차: Module 1-2
- 3-4주차: Module 3
- 5-6주차: Module 4
- 7주차: Module 5
- 8주차: Module 6 + 종합 평가

### 자기 주도 과정 (12주)
- 주 1회 모임, 개인 연습 병행
- 온라인 피드백 및 질의응답

## 📝 평가 기준

### 실기 평가 (70%)
| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| Tekiah | 15점 | 음색, 지속시간, 안정성 |
| Shevarim | 15점 | 3음의 균일성, 연결성 |
| Teruah | 15점 | 9음의 명확성, 리듬 |
| Tekiah Gedolah | 15점 | 지속시간(최소 9초), 음질 |
| 앙상블 조화 | 10점 | 시작/종료 동기화, 음량 조절 |

| 항목 | 배점 | 평가 기준 |
|------|------|----------|
| 성경적 지식 | 10점 | 쇼파르 관련 성경 구절 이해 |
| 전통 지식 | 10점 | 유대 전통과 의식 절차 |
| 실무 지식 | 10점 | 악기 관리, 행사 준비 |
