## 🔄 문제 해결 체크리스트

### 소리가 안 날 때

- [ ] 입술이 충분히 촉촉한가?
- [ ] 앙부슈르가 너무 긴장되어 있지 않은가?
- [ ] 공기가 충분한가?
- [ ] 마우스피스 위치가 적절한가?
- [ ] 악기에 막힘이 없는가?

### 소리가 불안정할 때

- [ ] 호흡이 일정한가?
- [ ] 앙부슈르가 유지되고 있는가?
- [ ] 악기를 안정적으로 잡고 있는가?
- [ ] 긴장하고 있지 않은가?
- [ ] 충분히 휴식했는가?

### 삐걱거리는 소리가 날 때

- [ ] 공기를 너무 세게 불고 있지 않은가?
- [ ] 앙부슈르가 너무 조여 있지 않은가?
- [ ] 마우스피스 위치를 조정해 보았는가?
- [ ] 입술 버징부터 다시 연습해 보았는가?

### 오래 유지가 안 될 때

- [ ] 횡격막 호흡을 하고 있는가?
- [ ] 공기를 효율적으로 사용하고 있는가?
- [ ] 불필요한 긴장이 있지 않은가?
- [ ] 충분한 호흡 연습을 했는가?
