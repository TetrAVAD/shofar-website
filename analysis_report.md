# Shofar Academy 시스템 분석 보고서

## 현재 상태 확인 결과 (2024-01-14)

### 확인된 작동 기능
1. **학습 시작하기 버튼**: 클릭 시 "학습을 시작합니다!" 토스트 메시지 표시됨 - 작동함
2. **모듈 상세 페이지**: MD 파일 기반 텍스트 콘텐츠가 잘 표시됨
3. **사이드바 네비게이션**: 모듈 1-6, 공지사항, 자유게시판 메뉴 정상 작동
4. **체크리스트**: 4개 항목 표시됨 (쇼파르의 어원과 의미 이해, 성경적 역사 타임라인 파악 등)

### 핵심 문제점
1. **로그인/회원가입 시스템**: 현재 Manus OAuth만 지원 - Gmail 로그인 미지원
2. **비회원 접근 처리**: 비회원도 모든 콘텐츠에 접근 가능 - 회원가입 유도 없음
3. **홈페이지 구조**: 현재 대시보드 형태 - 텍스트 교육 형태로 변경 필요

### 사용자 요구사항
1. Gmail 계정 로그인으로 쉽게 회원가입
2. 비회원 접근 시 화면 가운데에 회원가입 안내 버튼 표시
3. MD 파일 기반 텍스트 교육 형태로 홈페이지 재구성

### 기술적 제약사항
- 현재 템플릿은 Manus OAuth 기반 (Google OAuth 별도 설정 필요)
- Google OAuth를 위해서는 Google Cloud Console에서 OAuth 클라이언트 ID 발급 필요
